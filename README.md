# Auditor-a-de-Hid-Rasberry-Pi-Pico
Proyecto de automatización USB HID que utiliza Raspberry Pi y CircuitPython para la auditoría controlada de Windows y la recopilación de inventario mediante inyección de teclado basada en PowerShell, centrado en la estabilidad, el manejo del diseño y la ingeniería de carga útil reproducible.
